# 🌦️ Weather Dashboard

A modern and responsive **Weather Dashboard** built using **HTML, CSS (Tailwind), and JavaScript**.
It allows users to search for any city or use their current location to get real-time weather details.

---

## 🚀 Features

* 🔍 Search weather by city name
* 📍 Get weather using current location (Geolocation API)
* 🌡️ Real-time temperature and "feels like" data
* 💨 Wind speed, humidity, pressure, and visibility
* 🌅 Sunrise & 🌇 Sunset timings
* 🎨 Beautiful modern UI with gradient design
* ⚡ Fast and responsive

## ⚙️ Setup & Installation

### 1️⃣ Clone the repository

```bash
git clone https://github.com/yourusername/weather-dashboard.git
cd weather-dashboard
```

---

### 2️⃣ Install dependencies

```bash
npm install
```

---

### 3️⃣ Run Tailwind build

```bash
npx tailwindcss -i ./src/input.css -o ./src/output.css --watch
```

---

### 4️⃣ Open project

Just open:

```
frontend/index.html
```

in your browser 🚀

---

## 🔑 API Setup

1. Go to **OpenWeatherMap**
2. Generate your API key
3. Add it inside your `index.js` file:

```js
const API_KEY = "your_api_key_here";
```
